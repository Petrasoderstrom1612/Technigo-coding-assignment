So you wonder what exactly is HTTPS?

HTTPS (Hyper Text Transfer Protocol Secure) is an encrypted version of the HTTP protocol.

HTTPS stands for Hyper Text Transfer Protocol Secure. It is the request respons mechanism enabling the communication between a client and a server while using the safety layer TLS - Transport Layer Security or so-called SSL - Secured Sockets Layer (used interchangeably). The communication is encrypted by a public key from the server’s side and facilitated by a session key generated at the client’s side. The security of the server is ensured via a digital certificate validated by the trusted authority leaving clients at ease when entering sensitive data, especially valuable when facilitating payment transactions on the web.
